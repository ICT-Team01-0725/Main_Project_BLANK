01 = 대시보드
   : 여러 정보들을 간략하게 볼 수 있도록
   ex) 회원 목록, 문의 내역, 공지사항,팝업, 뉴스 피드 등

02 = 사용자 관리
   01) 일반 사용자
      : 관리자가 사용자들의 대한 정보를 확인하거나 제재를 가할 수 있게 설정 (신고, 경고, 회원조회 등)
   02) 전문 사용자
      : 일반 사용자와 전문 회원에 대한 정보를 따로 구성해 각각 다른 페이지에서 확인 가능(전문가 자격증 제출 및 승인 + 신고, 경고, 정지)

03 = 콘텐츠 관리
   : 설명문 양식의 페이지는 관리자가 변경 가능하게 설정
   : 각종 DB들이 변경되었을 때 그 DB들에 대해 수정할 수 있도록

04 = 공지사항 질문/답변 및 게시판 관리
   : 커뮤니티에 대한 제재나 각종 게시판에 CRUD 가능
   , 단 약사 상담의 경우 관리자가 댓글 작성은 못하도록
   , 팝업창의 관리

05 = 데이터 분석/ 통계
   : 접속자 수 카운팅, 조회수 및 클릭률 등의 데이터 분석
    트래킹

06 = 보안/ 권한관리
   : root계정이 다른 관리자 계정 생성 및 관리 등 모든 권한 O
   : root계정은 계정 탈퇴버튼 X
   : 관리자 계정들은 일반유저 및 전문가 유저 관리 및 각종 게시판 관리
   : 전문가 유저 등록을 위한 약사 및 의사 면허 첨부파일 확인


-------------------


00 = 대시보드
1… 전체 요약 정보
- 주요 통계 및 실시간 데이터 (회원 수, 접속자 수, 클릭률 등)
2… 회원 활동 현황
- 최근 가입 회원, 문의 내역, 공지사항
3… 콘텐츠 및 알림 관리
- 팝업, 뉴스 피드, 중요한 알림

01 = 사용자 관리
1… 일반 사용자 관리
- 사용자 정보 조회 및 검색
- 제재 기능 (신고, 경고, 정지)
- 회원 활동 기록 열람
2… 전문가 사용자 관리
- 전문가 정보 조회 및 검색
- 자격증 제출 및 승인 관리
- 전문가 활동 모니터링 및 제재 기능
- 일반 사용자와 전문가 사용자 구분

02 = 콘텐츠 관리
1… 데이터베이스 관리 및 보수
- 데이터 수정 및 업데이트
- DB 백업 및 복구 기능
2… 서브페이지 수정
- 설명문 및 페이지 콘텐츠 편집
- 페이지 레이아웃 관리 및 미리보기

03 = 게시판 관리
1… 커뮤니티 게시판 관리
- 게시글 및 댓글 CRUD (생성, 조회, 수정, 삭제)
- 게시판별 접근 권한 설정
- 커뮤니티 규칙 위반 제재
2… 팝업 관리
- 팝업 내용 추가 및 수정
- 팝업 표시 설정 (기간 및 위치 지정)

04 = 데이터 분석 / 통계
1… 카테고리별 클릭률 분석
- 각 콘텐츠 및 기능별 클릭 통계
2… 회원 수 증감 분석
- 월별, 분기별 회원 증가 및 감소 그래프
3… 접속 및 활동 분석
- 접속자 수, 페이지 조회수, 사용자 활동 트래킹

05 = 보안 / 권한 관리
1… 로그 관리
- 관리자 및 사용자 활동 로그 열람
- 로그인 이력 및 IP 기록
2… 관리자 계정 관리
- 일반 관리자 계정 생성, 수정, 삭제 (슈퍼관리자 권한)
- 관리자 권한 설정 및 관리
3… 전문가 등록 관리
- 면허 첨부파일 확인 및 승인 (약사, 의사 등)
4… Root 계정 관리
- 전체 권한 보유 (탈퇴 불가)
- 모든 관리자 및 사용자 계정에 대한 최상위 관리
